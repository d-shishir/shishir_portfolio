# Shishir Lamichhane - Personal Portfolio

Welcome to the repository for my personal portfolio website. This project is designed to showcase my skills, professional experience, projects, and contact information, as well as serve as a hub for my professional online presence.

## 🚀 Live Demo

[Visit my Portfolio Website](https://d-shishir.github.io/shishir_portfolio/)

## ✨ Features

- **Interactive Hero Section:** Features a dynamic, custom HTML5 Canvas interactive wipe effect and clean typography.
- **Dedicated Pages:** Minimalist and responsive internal pages for About, Projects, Contact, and a dedicated Resume view.
- **ATS-Friendly PDF Export:** The Resume page includes a "Download PDF" function which aggressively recompiles the active page into a clean, text-centric, strictly A2 2-page format highly optimized for Applicant Tracking Systems (ATS).
- **Responsive Design:** fully responsive across all breakpoints, ensuring optimal viewing on mobile devices, tablets, and large desktop screens.
- **Fluid Animations:** Powered by `framer-motion` for smooth page transitions and element reveals.

## 🛠️ Tech Stack

- **Frontend Framework:** React 19
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** CSS3 (Vanilla / Custom Properties) + Tailwind CSS 4
- **Routing:** React Router v7
- **Animations:** Framer Motion
- **Icons:** Lucide React & Tabler Icons

## 💻 Local Development

To run this project locally on your machine, follow these steps:

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- `npm` or `yarn`

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/d-shishir/shishir_portfolio.git
   cd shishir_portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.

### Build & Deploy

This project is configured to be deployed using GitHub Pages.

- **Create a production build:**
  ```bash
  npm run build
  ```

- **Deploy to GitHub Pages:**
  ```bash
  npm run deploy
  ```
